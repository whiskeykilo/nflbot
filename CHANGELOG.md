# Changelog

All notable changes to this project are documented in this file.

## v0.1.0 — 2025-08-30

- First public release for NFL spreads.
- Hard Rock spreads integration with American odds.
- Pinnacle alt-line reference with de‑vig fair probabilities.
- EV computation (with pushes) and Kelly sizing, unit cap.
- Discord webhook notifications with compact, readable format.
- SQLite signal ledger with de‑duplication.
- Optimized in‑app polling schedule; legacy schedule fallback.
- RUN_ONCE mode and quota handling for The Odds API.
